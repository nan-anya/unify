각 집단의 문제들
1. 경제적 문제(물자부족 등)
2. 외적이 있음(야생동물, 야만인 등)
3. 환경적 문제(물, 대지오염 등)
4. 탐관 오리(반란군 등)

역사적 배경 : 핵전쟁 약 70년 후 모든 국가의 정부가 붕괴하고 인구 수가 1/100로 줄어들어 인류는 더 작은 단위의 집단 생활을 시작.
환경의 파괴와 돌연변이의 출현으로 각 집단간의 연결이 미약함.
이 때 큰 두 개의 새력이 등장하면서 옛날처럼 큰 힘을 가진 나라를 복구하려고 시도함.
두 큰 세력의 이름은 미정 A,B로 호칭

A세력 : 군사력보다는 경제력에서 강함. 지도자의 권력이 상대적으로 약하고 내각(?)의 힘이 강함. 지도자는 항상 숙청이나 암살의 위험에 노출되어있음.

파견관 : A세력에서 다른 집단들을 병합하기위해 파견하는 관리. 그 세력의 지지를 얻는 것이 주임무. 생존률이 매우 낮음.

B세력 : 전제정권으로써 지도자의 권력이 막강함. 항상 군사력으로 주변을 정복하려고 하고있음.

주인공 : A세력 지도자의 아들로서 성인이 되기까지 불안한 나날을 보냄. 그것이 싫어서 어떻게든 밖으로 나가기 위한이유반, 내각의 압박(간접적인 사살?)반 으로 파견관이 됨. 언변이 뛰어남.


집단a : 상당한 전쟁전 무기가 있어 안보에 도움이 되고있지만 공업생산력(기술력?)이 부족하여 생필품이나 농기구등의 수급이 어려움.

상황
농기구, 사냥도구의 품질이 좋지않음 -> 식량생산량이 적음 -> 식량부족 -> 인구부족 : 세력A과 무역을 통해 해결

정보
여관(주막?)에 음식을 주문했지만 변변한 것을 먹지못함. (단초)
굶어죽기 직전의 아이를 업고 동냥을 다니는 여인. (심각성 강조)
농기구가 부러져 직접 수리하고있는 농민. (힌트1)
차후 추가





집단b : 인구도 적당하고 나름대로 살만한환경을 가짐. 하지만 주변에 돌연변이 서식지가 있어 외부에서 접근하기를 꺼려함 외부와 단절되고 항상 불안한 삶을 살고 있음. 과거의 무기가 전무하다시피함. 돌연변이에게 목숨을 잃는 사람들이 많음.

상황
돌연변이들이 주기적으로 마을을 습격함 -> 군사력이 부족함 -> 무기필요 : 세력A가 집단a와 집단b사이의 무역을 중계함. 혹은 세력A의 군사력을 지원

정보
마을 외곽에 무리에서 떨어진 돌연변이 한마리.
병사들이 원시적인 무기로 무장하고 있음.
돌연변이에게 남편을 잃은 아내.
